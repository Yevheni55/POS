Atelier framework · Multi-tenant POS design system. Tento súbor je demo — production rozdelí base tokens (immutable) a per-tenant theme overlay. Theme picker hore prepína data-theme atribút na <html>. 6 themes pripravených (Lighthouse / Espresso / Greenroom / Neon Lab / Daylight / Lake), pridanie ďalšieho = ~30 riadkov CSS.

Toto je iba návrh. Implementácia v reálnej kase ide po surfacoch: tokeny.css → theme presets → komponenty → page-by-page roll-out.
